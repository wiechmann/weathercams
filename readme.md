# Hamburger Schietwetter

This repo contains two versions of the app:

- `weathercams/` – Version 1 (Dealing with reality)
- `x-mas-edition/` – Version 2 (Dream big)

## How to Run Locally

1. **Download or clone the repository**

    ```bash
    git clone https://github.com/wiechmann/weathercams.git
    ```

2. **Open the HTML file**

   *You can open it directly in a browser:*

   - Double‑click `index.html`.
   - Or drag the file into an open browser window.

3. **(Optional) Use a local server**

   - **Python 3.x**
    ```
    # in the app folder
    python3 -m http.server 8000
    ```
   Then open `http://localhost:8000` in your browser.
